Demo: min-width vs max-width

Acest exemplu arată cum se comportă două cutii în interiorul aceluiași container când folosim min-width sau max-width. Resizează fereastra browser-ului pentru a vedea ce se întâmplă când containerul devine mai mic sau mai mare.

min-width

width: 100% min-width: 240px
Min width: nu scade mai jos decât 240px
Când containerul este suficient de mare, cutia crește cu 100%. Când containerul devine prea mic, cutia rămâne cel puțin 240px lățime.

max-width

width: 100% max-width: 240px
Max width: nu crește mai mult de 240px
Când containerul este mare, cutia rămâne maxim 240px. Când containerul este mic, cutia se micșorează proporțional până la 0.

@media (max-width: 520px)

width: 100% background: albastru
Redu fereastra sub 520px pentru a activa media query
Această cutie își schimbă culoarea când browserul devine mai îngust decât 520px. Este un exemplu vizual de media query.

Diferența cheie

min-width fixează limita minimă de lățime. Cutia nu poate fi mai îngustă decât valoarea declarată. max-width fixează limita maximă. Cutia nu poate fi mai lată decât valoarea declarată.
