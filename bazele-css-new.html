Începe lecția

1. Formate de Culori în CSS

Moduri diferite de a specifica culori: nume, hex, rgb, hsl, hsla

De ce sunt importante culori?

Culori sunt fundamentale pentru designul web. Ele transmit emoții, crează contrast, îmbunătățesc readibilitatea și construiesc identitatea brand-ului.

Principii de bază: Contrast suficient, paletă consistentă, accesibilitate (WCAG), psihologia culorilor.

CSS oferă 6 moduri principale să specifici culori: nume, hex, rgb, rgba, hsl, hsla.

Comparație Formate de Culori

1. Nume de Culoare

Cele mai comune culori au nume predefinite.

color: red;
color: blue;
color: tomato;

✓ Ușor de citit
✗ Doar ~150 culori disponibile

2. Hex (Hexazecimal)

Format: #RRGGBB — 6 cifre hex reprezentând roșu, verde, albastru.

color: #FF6347;
color: #F63; /* scurt */
background: #0096C7;

✓ 16 milioane culori
✓ Compact

3. RGB (Red, Green, Blue)

Format: rgb(R, G, B) — valori 0-255 pentru fiecare culoare.

color: rgb(255, 99, 71);
background: rgb(0, 150, 199);
border: rgb(100, 100, 100);

✓ Ușor de manipulat
✓ Transparență cu rgba()

4. RGBA (RGB + Alpha)

Format: rgba(R, G, B, A) — A este opacitate 0-1.

background: rgba(255, 99, 71, 0.8);
box-shadow: rgba(0, 0, 0, 0.2);
color: rgba(50, 50, 50, 0.9);

✓ Transparență control total
✓ Perfect pentru overlay-uri

5. HSL (Hue, Saturation, Lightness)

Format: hsl(H, S%, L%) — mai intuitiv pentru designers.

color: hsl(0, 100%, 50%);
background: hsl(200, 100%, 50%);
border: hsl(120, 100%, 50%);

✓ Ușor de creat variații
✓ Gândire naturală

6. HSLA (HSL + Alpha)

Format: hsla(H, S%, L%, A) — HSL cu transparență.

background: hsla(200, 100%, 50%, 0.8);
color: hsla(0, 100%, 50%, 0.5);
border: hsla(120, 70%, 60%, 0.9);

✓ Cea mai flexibilă
✓ Transparență + control ușor

2. Background Properties

Controlează fondul elementelor — culori, imagini, pozitii, dimensiuni

background-color

Culoarea de fundal a elementului

background-color: #0096C7;
background-color: rgb(0, 150, 199);
background-color: rgba(0, 150, 199, 0.5);

background-image

Imagine ca fundal (url sau gradient)

background-image: 
  url('photo.jpg');
background-image: 
  linear-gradient(45deg, blue, red);

background-size

Dimensiunea imaginii de fundal

background-size: cover;
background-size: contain;
background-size: 200px 100px;

background-position

Poziția imaginii în cadrul containerului

background-position: center;
background-position: top left;
background-position: 50% 50%;

background-repeat

Comportament repeat al imaginii

background-repeat: repeat;
background-repeat: no-repeat;
background-repeat: repeat-x;

background-attachment

Fixează background-ul la scroll

background-attachment: scroll;
background-attachment: fixed;
background-attachment: local;

💡 Pro Tip: Folosește shorthand background pentru a combina mai multe proprietăți:

background: 
  url('bg.jpg') center / cover no-repeat fixed;

3. Gradiente CSS

Tranziții netede între culori — linear, radial, conic

Linear Gradient

Tranziție liniară între culori

background: 
  linear-gradient(
    to right,
    #667eea,
    #764ba2
  );

Radial Gradient

Radiază din centru spre exterior

background: 
  radial-gradient(
    circle,
    #f093fb,
    #f5576c
  );

Conic Gradient

Rotație în jurul unui punct

background: 
  conic-gradient(
    red, yellow, green,
    blue, red
  );

Multi-Color Linear

Gradient cu mai mult de 2 culori

background: 
  linear-gradient(45deg,
    #ff6b6b, #ffd93d,
    #6bcf7f, #4d96ff,
    #d946ef
  );

Repeating Gradient

Repetă gradientul

background: 
  repeating-linear-gradient(
    45deg,
    #FFD700 0px,
    #FFA500 20px
  );

Diagonal Gradient

Unghi 45 grade

background: 
  linear-gradient(135deg,
    #a8edea,
    #fed6e3
  );

4. Cazuri de Utilizare — Interactiv

Aplicații practice ale culorilor și background-urilor — selectează pentru a vedea codul

🎨 Hero Section — Fundal cu gradient

Utilizări: Landing pages, hero sections, motivație vizuală

Titlu Atractiv

Descriere sub titlu

.hero {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
  padding: 80px 40px;
  text-align: center;
}

🎴 Color Cards — Carduri cu culori diferite

Utilizări: Categorii, servicii, feature highlights

.card {
  background-color: #FF6B6B;
  padding: 20px;
  border-radius: 8px;
  color: white;
  transition: transform 0.3s;
}

.card:hover {
  transform: scale(1.05);
}

🔘 Gradient Button — Buton modern cu gradient

Utilizări: Call-to-action buttons, interactive elements

.btn-gradient {
  background: linear-gradient(45deg, #FF6B6B, #FFD93D);
  color: white;
  border: none;
  padding: 12px 32px;
  border-radius: 8px;
  cursor: pointer;
  font-weight: bold;
  transition: transform 0.2s;
}

.btn-gradient:hover {
  transform: translateY(-2px);
}

🌄 Image Overlay — Gradient peste imagine

Utilizări: Featured images, blog post covers, readability overlay

Titlu Articol

Descriere scurtă care se citește bine

.image-wrapper {
  position: relative;
  background-image: url('image.jpg');
  background-size: cover;
}

.image-wrapper::after {
  content: '';
  position: absolute;
  top: 0; left: 0;
  right: 0; bottom: 0;
  background: linear-gradient(
    180deg, rgba(0,0,0,0), rgba(0,0,0,0.7)
  );
}

✍️ Text pe Gradient — Contrast bun

Utilizări: Titluri eye-catching, promotional banners

Text Colorat pe Fundal Gradient

.text-on-gradient {
  background: linear-gradient(
    -45deg, #FF6B6B, #FFD93D, #6BCF7F
  );
  color: white;
  padding: 40px;
  text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
  text-align: center;
}

🎯 Gradient Border — Contur colorat

Utilizări: Special cards, highlighted sections, premium look

Premium Card

Acesta este un card special cu contur gradient

.gradient-border {
  background: linear-gradient(
    white, white
  ) padding-box,
    linear-gradient(
      45deg, #667eea, #764ba2
    ) border-box;
  border: 3px solid transparent;
  border-radius: 8px;
  padding: 16px;
}
