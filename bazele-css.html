1. Selectori CSS

Cum vizăm elementele pe care vrem să le stilizăm

⬛

Selector de element

Vizează toate elementele de acel tip.

p {
  color: red;
}
●

Selector de clasă

Vizează toate elementele cu acea clasă HTML.

.titlu {
  font-size: 24px;
}
#

Selector de ID

Vizează un singur element unic pe pagină.

#meniu {
  background: navy;
}
*

Selector universal

Vizează absolut toate elementele din pagină.

* {
  box-sizing: border-box;
}
↓

Selector descendent

Vizează elementele aflate în interiorul altui element.

nav a {
  color: white;
}
:

Pseudo-clasă

Vizează un element în funcție de starea sa.

a:hover {
  color: blue;
}
tr:nth-child(even) {
  background: #f0f0f0;
}

2. Box Model

Fiecare element HTML este o cutie cu 4 straturi

margin
border
padding
content
text, imagini...
margin Spațiu în exteriorul elementului. Separă elementele între ele.
border Conturul vizibil al elementului. Poate fi solid, dashed, dotted.
padding Spațiu interior, între border și conținut.
content Zona efectivă cu text, imagini sau alte elemente. 
.cutie {
  margin:  20px;        /* exterior */
  border:  3px solid red; /* contur */
  padding: 15px;        /* interior */
  width:   200px;        /* lățime */
}

12. Variabile CSS

:root + var() — definim valori reutilizabile

Definire în :root

:root {
  --culoare: #0096C7;
  --raza:    10px;
  --font:    sans-serif;
}

Utilizare cu var()

.buton {
  background:
    var(--culoare);
  border-radius:
    var(--raza);
  font-family:
    var(--font);
}

Avantaje

Schimbi o singură valoare și se actualizează tot site-ul.

Util pentru teme dark/light, culori de brand, spațieri consistente.
