1. Fundațiile Flexbox

Cum funcționează layoutul flexibil — axă principală și transversală

Ce este Flexbox?

Flexbox este un sistem de layout CSS care permite distribuirea spațiului și alinierea elementelor într-o dimensiune (pe o singură axă — rând sau coloană).

Când îl folosești: Navigații, butoane inline, bar-uri cu conținut distribuit, card-uri pe o linie, etc.

Avantajul principal: display: flex pe părinte + proprietățile copiilor = layout responsive fără media queries complexe.

Axe în Flexbox

→

Axa Principală (Main Axis)

Direcția în care se aranjează elementele. Default: orizontală (stânga → dreapta).

flex-direction: row (orizontal) sau column (vertical)

↓

Axa Transversală (Cross Axis)

Axa perpendiculară pe axa principală. Pentru alinierea verticală (în row mode).

align-items: aliniază toți copiii pe axa transversală

📋 Form Layout — Input + Button pe aceeași linie

Utilizări: Search bars, newsletter signup, input groups

.form-group {
  display: flex;
  gap: 8px;
}

.form-input {
  flex: 1; /* ia tot spațiul disponibil */
  padding: 10px;
  border: 1px solid #ddd;
  border-radius: 4px;
}

.form-btn {
  flex: 0 0 auto; /* nu crește */
  padding: 10px 24px;
}
